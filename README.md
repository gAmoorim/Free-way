#  Free way JS 🚗

> Um pequeno jogo desenvolvido em JavaScript inspirado no clássico Freeway, onde o objetivo é atravessar uma estrada cheia de carros em movimento.

O jogador controla um personagem que precisa atravessar a rua evitando os carros que passam constantemente.

---

## Como funciona

- O jogador controla o personagem para atravessar a estrada.

- Diversos carros passam pela pista com velocidades diferentes.

- Se o personagem colidir com um carro, ele volta para o início.

- O objetivo é chegar ao outro lado da rua.

---

## Objetivo do projeto

- Praticar JavaScript

- Trabalhar com lógica de jogos

- Utilizar bibliotecas gráficas como p5.js

- Aprender conceitos básicos de colisão e movimentação
